## 随译
基于鸿蒙3.1/4.0，使用Stage 模型和 ArkUI 开发
由于身边没有对应的设备，只在MetaPadPro 2019 鸿蒙4.0.0上验证

### 技术说明

#### 翻译api 

[x] 百度翻译开放平台：https://fanyi-api.baidu.com/
[] 百度机器翻译
[] 有道翻译

#### 状态管理

使用了鸿蒙 ArkUI 自身的状态管理方案

* PersistentStorage 存储用户偏好设置，比如翻译时机，api 所需要的 key 等
